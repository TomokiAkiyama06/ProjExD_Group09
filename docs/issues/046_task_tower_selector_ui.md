## 概要

プレイヤー1がどのタワー種類を設置しようとしているか視覚的にわかるUIを実装する。

## 作業

- 数字キー1〜4 で属性タワーの切替（基本機能のスタブを実装で埋める）
- 画面下部または上部に4種類のタワーアイコンを並べ、現在選択中のものをハイライト
- マウス位置にゴーストプレビュー（半透明のタワーを表示）
- 設置不可エリアでは赤色に変える

## 完了条件

- プレイヤーがどのタワーを設置中か視覚的にわかる
- 設置できる場所がプレビューで分かる
