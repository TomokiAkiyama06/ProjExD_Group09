## 概要

前線役プレイヤーが武器を切り替えられるUIとキー入力を実装する。

## 作業

- 数字キー（例：Q/E、または1/2/3）またはマウスホイールで武器切替
- 画面の前線役近辺に現在の武器アイコンを表示
- 切替時に短いクールタイム（連打防止）

## 完了条件

- プレイ中に武器を切替できる
- どの武器を装備中かUIで一目でわかる
