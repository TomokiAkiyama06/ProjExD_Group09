## 概要

プレイヤー1（建築役）の操作（マウス＋数字キー）を実装する。タワー設置や種類切替を行う。

## 作業

- マウス左クリック：タワー設置（リソース消費、設置可能エリアの判定）
- マウス右クリック：タワー選択UIを開く（アップグレード／売却の枠だけ用意。中身は担当③）
- 数字キー 1〜4：設置するタワー種類の切替（属性タワーは担当③で実装、ここでは枠だけ）
- スペースキー：次ウェーブの早期開始

## 完了条件

- soloモードでマウス左クリックでタワーを設置できる
- 数字キーで設置するタワー種類を切り替えられる
